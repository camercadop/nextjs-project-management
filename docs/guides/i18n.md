# How To: Internationalization

## Add a New Translation Namespace

Namespaces map to JSON files in `public/locales/{lng}/`.

### 1. Create the JSON files for each locale

```
public/locales/es/<namespace>.json
public/locales/en/<namespace>.json
```

Example for a `task` namespace:

```json
// public/locales/es/task.json
{
    "task.title_label": "Titulo",
    "task.create_success": "Tarea creada exitosamente"
}
```

```json
// public/locales/en/task.json
{
    "task.title_label": "Title",
    "task.create_success": "Task created successfully"
}
```

### 2. Use in Server Components

```tsx
import { t } from '@/lib/i18n-server'

export default async function TaskPage() {
    return <h1>{t('task.title_label', 'task')}</h1>
}
```

### 3. Use in Client Components

```tsx
import { useTranslation } from 'react-i18next'

export function TaskForm() {
    const { t } = useTranslation('task')
    return <label>{t('task.title_label')}</label>
}
```

## Add a New Supported Language

### 1. Create the locale directory

```bash
mkdir -p public/locales/<lng>
```

### 2. Copy and translate all namespace files

```bash
cp public/locales/en/*.json public/locales/<lng>/
```

Then translate all values in the copied files.

### 3. Update the environment variable

Add the new language code to `NEXT_PUBLIC_SUPPORTED_LOCALES`:

```env
NEXT_PUBLIC_SUPPORTED_LOCALES=es,en,fr
```

No code changes are needed — the i18n config reads supported languages from this variable.

## Key Naming Rules

- Use `lowercase_snake_case` for all keys: `issue.status_in_progress`
- Prefix keys with the namespace: `task.create_success`
- Group related keys logically
- Always add keys to ALL supported locale files simultaneously

## Server vs Client Usage

| Context | Import | Signature |
|---------|--------|-----------|
| Server Components | `import { t } from '@/lib/i18n-server'` | `t(key, ns?, lng?)` |
| Client Components | `import { useTranslation } from 'react-i18next'` | `const { t } = useTranslation(ns)` |

The `ns` parameter defaults to `'common'` if omitted.
